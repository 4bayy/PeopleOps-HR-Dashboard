# PeopleOps - Employee Management Dashboard

A modern and responsive Employee Management Dashboard built with **React**, **Vite**, **React Router**, and **Tailwind CSS**. The application provides an intuitive interface for managing employees, viewing employee profiles, and monitoring HR-related information.

## Live Demo

🚀 **Deployed on Vercel:**  
https://people-ops-hr-dashboard.vercel.app/

---

## ✨ Features

### Dashboard
- Responsive dashboard layout
- Employee statistics cards
- Attendance overview
- Department distribution chart
- Quick action cards

### Employee Management
- Employee list (Mobile & Desktop)
- Employee profile page
- Add Employee form
- Department & Employment details
- Profile photo upload
- Responsive employee cards
- Desktop table view

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/your-username/peopleops-dashboard.git
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
